# AI Novel Writer — Web UI 优化设计方案

## 概述

对现有 AI Novel Writer 前端进行全面的 UI/UX 优化，覆盖 6 大方向：设计系统、侧边栏功能、数据可视化、交互动效、编辑器体验、创建流程。

## 1. 设计系统

### 配色方案
- **主色 (Primary)**: `#6366F1` (Indigo) — 按钮、链接、激活态
- **主色深色 (Primary Dark)**: `#4F46E5` — 悬停态
- **主色浅色 (Primary Light)**: `#EEF2FF` — 背景、轻量高亮
- **辅色 (Secondary)**: `#06B6D4` (Cyan) — 进度条、装饰元素
- **成功 (Success)**: `#10B981` — 完成态
- **警告 (Warning)**: `#F59E0B` — 进行中
- **危险 (Danger)**: `#EF4444` — 删除/错误
- **侧边栏 (Sidebar)**: `#0F0D2E` 深色渐变背景

### 字体
- **标题**: Inter Semi Bold (700)
- **正文**: Inter Regular (400), line-height 1.7
- **辅助文本**: Inter Regular, color `#64748B`

### 圆角与阴影
- **圆角**: 卡片 12px, 按钮 8px, 标签 20px
- **阴影**: `shadow` (基础), `shadow-lg` (悬停), `shadow-xl` (特殊态)

## 2. 布局重构

### 侧边栏 (AppLayout.tsx)
- 背景从白色改为深色渐变 `#0F0D2E → #1A1744`
- 增加用户头像（渐变色圆形 + 绿色在线点）
- 菜单项带左侧激活指示条（3px 竖线）
- 增加"最近项目"快速切换区域（3个最近项目 + 时间戳）
- 底部统计概览（项目数/章节数/字数）
- 菜单项支持数量徽标

### 顶部工具栏
- 毛玻璃效果 (backdrop-filter: blur + rgba 背景)
- 搜索框（全局搜索项目、章节）
- 暗色模式切换 Toggle
- 通知铃铛图标
- 用户头像

## 3. 仪表盘 (Dashboard.tsx)

### 统计卡片
- 卡片增加彩色图标背景（不同色系）
- 趋势指标（↑ 本月新增 / 本周生成）
- 迷你走势图（7 个小柱状条）
- 各卡片独立色系：书籍(Indigo)、章节(绿)、任务(黄)、字数(粉)

### 写作趋势图
- 双维度柱状图：章节数(Indigo) + 字数(Cyan)
- 悬停显示数值
- 较上周变化百分比
- 🔥 标记峰值日

### 写作热力图
- GitHub-style 3周活动日历
- 4级深浅表示写作量
- 今日高亮边框
- 连续写作天数统计 + 最佳日 + 日均统计

### 最近活动
- 时间线组件显示生成记录
- 成功/进行中/失败三种状态

## 4. 项目详情 (BookDetail.tsx)

### 章节列表
- 从 Ant Design Table 改为自定义卡片式布局
- 左侧序号圆圈（悬停变为主色填充）
- 右侧操作按钮默认隐藏，悬停显示
- 左侧颜色边框表示状态（绿=完成, 蓝=进行, 黄=生成中, 灰=待定）
- 生成中的章节带脉冲动画

### 拖拽排序
- 左侧⠿ 手柄图标
- `cursor: grab / grabbing`
- 拖拽时缩放 + 阴影增强

### 批量操作
- "批量操作"按钮切换模式
- 切换后拖拽手柄隐藏，checkbox 显示
- 批量删除 / 批量导出

### 生成进度条
- 旋转 spinner + 脉冲动画
- AI 来源 + 预计时间显示
- 取消按钮
- 已写字数实时更新

### 项目设置面板
- 输出格式改为按钮组（MD/TXT/DOCX）
- 定时生成一行排列：开关 + 时间 + 每天 + 章数
- 生成状态条：今日已生成 X/X 章

### 主题配置
- 毛玻璃卡片容器
- Tab 增加"视角"、"时间线"、"高级"选项
- 语气标签选择器

## 5. 章节编辑器 (ChapterEditor.tsx)

### 工具栏
- 撤销/重做
- 加粗/斜体/下划线/删除线
- 标题 H1/H2/H3
- 列表（有序/无序/待办）
- 链接/图片/引用插入
- 预览模式切换
- 保存 + 重新生成按钮
- 各按钮带 Tooltip 提示

### 状态栏
- 自动保存指示（绿色圆点）
- 字数统计：当前 / 目标
- 写作计时
- AI 来源显示
- 专注模式入口

### 右侧面板
- **大纲结构**：章节列表，当前章节高亮，可点击跳转
- **写作目标**：圆形进度卡片，目标达成百分比 + 剩余字数
- **快捷操作**：导出本章 + 查找替换

### 编辑器主体
- 标题可编辑 (contenteditable)
- 正文内容行高 1.9
- 专注模式隐藏侧栏

## 6. 创建向导 (BookList.tsx 新建弹窗升级)

### 分步流程
- 4 步：基本信息 → 主题设定 → AI 配置 → 预览完成
- 顶部步骤指示器（完成=绿色✓ / 进行中=主色 / 待办=灰色）
- 步骤间连接线

### Step 1: 基本信息
- 书名、作者、简介、输出格式
- 直接集成在原弹窗中，改为向导卡片

### Step 2: 主题设定
- 小说类型（下拉选择）
- 写作风格（自由输入）
- 语调（标签选择器）
- 目标字数/章
- 故事简介

### Step 3: AI 配置
- AI 来源选择
- 模型参数

### Step 4: 预览
- 摘要信息确认
- 创建按钮

## 7. 通用组件

### 按钮系统
- `btn-primary` / `btn-outline` / `btn-ghost`
- `btn-sm` / `btn-xs` / `btn-icon`
- `btn-group` 按钮组（MD|TXT|DOCX 样式）

### 标签 (Tag)
- `tag-green` / `tag-blue` / `tag-orange` / `tag-purple` / `tag-red`
- 用于状态显示和选择器

### 输入框
- `input-modern` 统一风格
- focus 状态：主色边框 + 3px 阴影光晕

### Tooltip
- 悬停时从底部滑入
- 深色背景白色文字

### Skeleton 骨架屏
- shimmer 渐变动画（200% 背景偏移）

### 空状态
- 居中插画 + 文字引导 + 操作按钮

## 8. 技术实现说明

### 样式方案
- 使用 Ant Design ConfigProvider 的 `theme.token` 自定义主题色
- 自定义 CSS 类覆盖 Ant Design 默认组件样式
- 全局 CSS 变量统一管理配色

### 组件改动清单 (前端文件)
| 文件 | 改动 |
|------|------|
| `AppLayout.tsx` | 侧边栏重构 + 顶部工具栏 + 暗色切换 |
| `Dashboard.tsx` | 统计卡片升级 + 趋势图 + 热力图 + 时间线 |
| `BookList.tsx` | 新建弹窗改为分步向导 |
| `BookDetail.tsx` | 章节卡片 + 拖拽 + 批量 + 生成进度 |
| `ChapterEditor.tsx` | 富文本工具栏 + 状态栏 + 大纲面板 |
| `types/index.ts` | 新增类型定义 |
| `api/*.ts` | 新增批量操作等 API |
| `index.css` | 新增全局样式变量和自定义类 |

### 依赖新增
- 拖拽排序：`@dnd-kit/core` + `@dnd-kit/sortable` (或简单 HTML5 Drag API)
- 可编辑内容：`contentEditable` (原生) 或 `react-quill` / `slate.js`
- 图表：纯 CSS 实现（无第三方图表库依赖）

## 9. 优先级排序

### P0 (必须实现)
1. 设计系统 CSS 变量 + Ant Design 主题配置
2. 侧边栏深色重构
3. 仪表盘统计卡片升级（迷你图 + 趋势）
4. 章节卡片列表 + 生成进度动效
5. 编辑器状态栏（字数 + 保存状态）

### P1 (重要)
1. 数据可视化（趋势图 + 热力图）
2. 创建向导分步流程
3. 毛玻璃顶部工具栏 + 搜索框

### P2 (锦上添花)
1. 拖拽排序
2. 批量操作
3. 暗色模式切换
4. 写作热力图
5. Tooltip + Tooltip 提示系统
6. 骨架屏加载态

## 10. 里程碑

1. **M1**: 设计系统 + 侧边栏 + 仪表盘升级
2. **M2**: 项目详情页重构（卡片 + 动效 + 生成进度）
3. **M3**: 编辑器升级 + 创建向导
4. **M4**: 数据可视化 + 通用组件库完善
5. **M5**: 暗色模式 + 拖拽 + 批量操作（P2 功能）
